### Overview

Status codes are broken up into different classes:
- 100 - Informational
- 200 - Success Responses
- 300 - Redirection
- 400 - Clientside errors
- 500 - Serverside errors

### Common Status Codes

200 Response:
| Status Code | Description |
| -------------- | --------------- |
| 200 | Success |
| 201 | Resource created |
| 202 | Accepted |

400 Response:
| Status Code | Description |
| -------------- | --------------- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |

500 Response:
| Status Code | Description |
| -------------- | --------------- |
| 500 | Internal Server Error |
| 501 | Not Implemented |
| 502 | Bad Gateway |
| 503 | Service Unavailable|
| 504 | Gateway Timeout |

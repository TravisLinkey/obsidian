## Overview

A tree-based data structure that satisfies the **heap property**.

**Heap Property** - All parent nodes are less than/ greater than (min heap/ max heap) children nodes.

| Operation   | Time Complexity    |
|--------------- | --------------- |
| Insert   | O(log n)   |
| Remove   | O(log n)   |
| min   | O(1)   |
| heapify (build heap)   | O(n)   |

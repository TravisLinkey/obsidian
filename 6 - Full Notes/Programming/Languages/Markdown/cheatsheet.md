Tags: [[Programming]]

## Heading
    h1 - '#'
    h2 - '##'
    h3 - '###'

## Fonts
    bold - '** {your bold text} **'
    italic - '* {your italic text} *'
    blockquote - '> {your blockquote}'

## Lists
    ordered list - type 'ordered list'
        1. first
        2. second
        3. third
    unordered list - type 'unordered list'
        - first
        - second
        - third

## Code 
    code - type 'code' 
    codeblock - type 'codeblock'
        ```language
        
        ```
## Link
    link - [title](url)
    image - [alt text](image.jpg)
    
## Formatting
    table - type 'table'
        | Column1 | Column2 | Column3 |
        | ------------- | -------------- | -------------- |
        | Item1 | Item1 | Item1 |

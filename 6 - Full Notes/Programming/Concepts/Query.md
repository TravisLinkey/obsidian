## Overview

**Definition** - A request to a database for information.

Used for targeted retrieval in a table or a secondary index using only the primary key attributes.

### Details

Typically they are much faster, less resource-intensive and incur lower costs compared to using a [[Scan]] because they are more selective and retrieve fewer items.
